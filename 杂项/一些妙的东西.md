#### 枚举二进数的子集

对于二进制数 $x$​，可以用令 $k=x$，然后不断 $k=(k-1)\&x$​ 即可遍历 $x$ 的所有子集！



#### 无取模求组合数

用于防炸longlong：

```c++
ll C(int n, int m){
    ll r = 1;
    for(int i=1;i<=m;i++) r = r * (n-m+1) / i;
    return r;
}
```

式子不难推出，在除 $i$ 的时候，已经乘上了至少 $i$ 个连续的整数，其中必有 $i$ 的倍数（循环），故必能整除
